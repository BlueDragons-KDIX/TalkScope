# order-002: プレゼンレイヤのテスト／設定／リセット／デモ重要語の追加

## 日付
2026-05-02

## 概要
`presentation/` を主エントリとする構成で、従来 `app/App.tsx` にあった検証用機能と表示設定を復元・整理する指示の要約である。

## 指示・要望（要約）

| # | 内容 |
|---|------|
| 1 | デモテキスト・ライブデモを復活させる。ボタンはウィンドウ外とし、画面上部（発表終了ボタン付近）に「テスト」からポップアップで開く |
| 2 | アプリ内テーマとダークモードを、画面上部の別ボタンからモーダルで設定できるようにする |
| 3 | テストポップアップに「デモ重要単語マーキング」ON/OFF を追加する。サーバー未接続時はコード上のモック語リストのみでハイライト・バブルを試せるようにする |
| 4 | 文字起こしウィンドウのリセットをツールバー（テストボタンの横）へ移し、押下ですべてのウィンドウ内容をリセットする |
| 5 | `Frontend/AGENTS.md` に沿って必要なドキュメントを更新する |

## 実装方針（反映済み）

- デモのデータ・アルゴリズムは `debug/demo/`、ツールバーと同期フックは `presentation/`（詳細は ADR-007）。
- グローバルリセットは `termStore.resetSession()` とバブル・文字起こしのクリアで統一。
