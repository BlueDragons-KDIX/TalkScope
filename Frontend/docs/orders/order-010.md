# order-010: バブル自動切り替えボタンをヘッダーへ移動

## 日付

2026-05-29

## 参加者

- ユーザー（担当者）
- Codex

---

## 指示要約

バブルウィンドウの **自動切り替え ON/OFF** を、設定ポップアップ内から **ウィンドウヘッダー**（設定ボタンの左隣）へ移す。切り替え間隔のスライダーは設定ポップアップに残す。ON/OFF が一目で分かるよう強調表示する。

---

## 実装方針

1. **`BubbleAutoSwitchHeaderButton` を新規追加**
   - `termMapWindowSettingsStore` の `autoSwitchEnabled` をトグル。
   - ON: アクセント半透明背景・グロー・`ON` バッジ。OFF: 控えめなグレー・`OFF` バッジ。
   - `RefreshCw` アイコン + 「自動」ラベル。ON 時はゆっくり回転アニメーション。

2. **`LayoutEngine`**
   - `windowId === 'bubbleCloud'` のときのみ、設定ボタン左に配置。
   - `stopPropagation` でヘッダードラッグと干渉しないようにする。

3. **`WindowSettingsPanel`**
   - 自動切り替えトグル UI を削除。
   - 「切り替え間隔」スライダーのみ残す。

---

## 完了条件

- バブルヘッダーから ON/OFF を切り替えられ、バブル表示が従来どおり連動すること
- 間隔スライダーは設定ポップアップから変更できること
- 他ウィンドウのヘッダーに自動切り替えボタンが出ないこと
