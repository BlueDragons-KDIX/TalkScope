# order-016: IT単語フィルター閾値とバブルサイズの感度調整

## 日付

2026-05-29

## 指示要約

1. 重要単語（IT）フィルターの **初期閾値を 0.42** にする。
2. フィルター通過後のスコアはおおむね **0.42〜0.45** に集まるため、バブル半径はこの差でも段階的に変わる計算にする。
3. 重要度レベル: `round(score × 100) - 41`（0.42 → 1、0.43 → 2 …）。クリック・再出現は **+0.01**（レベル +1 相当）。

## バブルサイズの上限（参考）

| 項目 | 値 |
|------|-----|
| 設計上の運用最大レベル | **8**（スコア目安 **0.49**、基準半径 **62px**） |
| レベル1の基準半径 | 20px |
| レベルあたりの加算 | +6px |
| スケール適用後の最大半径 | **110px**（それ以上のスコア・レベルはここでクリップ） |
| スコアのみで上限に達する目安 | レベル15前後（基準半径 104px → 倍率1で 110 にクリップ） |

最大スコアは 0.45 に限られないため、クリックや高スコアでレベル8まで段階的に拡大できるようにしている。

## 実装

| ファイル | 内容 |
|----------|------|
| `ScoreThresholdFilter.ts` | `DEFAULT_SCORE_THRESHOLD = 0.42` |
| `bubbleSizeFromScore.ts` | レベル → 半径（基準 20px、レベルごと +6px） |
| `BubbleCloud.tsx` | 上記ユーティリティで半径算出 |
| `DefaultScoreUpdateStrategy.ts` | `CLICK_DELTA` / `FREQUENCY_DELTA` = 0.01 |
