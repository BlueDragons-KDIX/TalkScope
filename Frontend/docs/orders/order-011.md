# order-011: クリック履歴ウィンドウのフォントサイズ設定

## 日付

2026-05-29

## 指示要約

クリック履歴（`history`）ウィンドウの設定ポップアップが空だったため、説明ウィンドウと同様に **フォントサイズスライダー** を実装する。

## 実装方針

1. `historyWindowSettingsStore` を追加（10〜24px、デフォルト 12px、`localStorage` 永続化）
2. `WindowSettingsPanel` に `history` 向けスライダーを追加
3. `HistoryWindow` → `HistoryPanel` へ `fontSizePx` を渡し、語名・説明行に反映（説明は語名 −2px）

## 完了条件

- 設定ポップアップに「履歴テキスト」スライダーが表示されること
- スライダー操作で履歴リストの文字サイズが変わること
- リロード後も設定が保持されること
